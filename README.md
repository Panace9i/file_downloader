## Зависимости
- PHP 5.5.0 и выше

## Установка
Используя Composer:<br/>
Создайте или дополните уже существующий файл зависимостей следующей конструкцией

```
{
  "require": {
    "panace9i/file-downloader": "dev-master"
  }
}
```
Выполните команду
```
composer install
```
